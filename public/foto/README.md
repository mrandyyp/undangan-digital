# Panduan Folder Foto

## Struktur Folder

```
public/foto/
├── header.jpg          - Gambar background header/cover
├── galeri/             - Folder untuk foto galeri
│   ├── foto-1.jpg
│   ├── foto-2.jpg
│   ├── foto-3.jpg
│   ├── foto-4.jpg
│   ├── foto-5.jpg
│   └── foto-6.jpg
```

## Cara Menggunakan

1. **Gambar Header/Background**
   - Upload gambar header dengan nama `header.jpg` di folder `public/foto/`
   - Format yang disarankan: JPG atau PNG
   - Ukuran yang disarankan: 1920x1080px atau lebih tinggi

2. **Galeri Foto**
   - Upload foto-foto galeri ke folder `public/foto/galeri/`
   - Beri nama file: `foto-1.jpg`, `foto-2.jpg`, dst.
   - Minimal 6 foto untuk tampilan optimal
   - Format yang disarankan: JPG atau PNG
   - Ukuran yang disarankan: 800x600px atau lebih tinggi

## Tips

- Gunakan foto dengan resolusi tinggi untuk hasil terbaik
- Pastikan nama file sesuai dengan yang ada di kode
- Foto akan ditampilkan secara otomatis bergantian setiap 4 detik
